# Raspotify
https://hub.docker.com/r/derkades/raspotify

This will allow you to stream Spotify from devices to a Raspberry Pi.

```
mkdir -p $HOME/docker/raspotify
```
```
cd $HOME/docker/raspotify
```
```
docker-compose pull
```
```
docker-compose up -d
```
